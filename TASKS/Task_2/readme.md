  Answer---
  - API Used: This project uses the CoinGecko API, specifically the simple/price endpoint for live prices and the market_chart endpoint for historical data.

  - Data Extracted: The dashboard extracts live USD prices, 24-hour percentage change, 24-hour trading volume for 8 cryptocurrencies, and 7-day historical price trends for BTC, ETH, and SOL.

  - Why Visualization is Useful: Visualizations like line and bar charts make it easy to track short-term trends, compare market activity across coins, and quickly interpret large crypto datasets for better analysis and decision-making.
